Sobre o Projeto

Este projeto foi desenvolvido como parte de um trabalho acadêmico da faculdade, com o objetivo de oferecer um ambiente de aprendizado acessível e interativo, focado na melhoria das habilidades tecnológicas dos usuários. A proposta principal é criar uma plataforma que disponibiliza uma série de vídeos educativos, tanto de minha autoria quanto de terceiros, com o intuito de facilitar o aprimoramento de habilidades no uso de ferramentas tecnológicas. É importante ressaltar que não sou um professor, mas um estudante buscando compartilhar recursos úteis para quem deseja aprender mais sobre tecnologia. Além disso, não obtenho nenhum ganho financeiro com este site.

Este projeto tem como objetivo não apenas oferecer um espaço para aprimorar as habilidades tecnológicas dos usuários, mas também apoiar os criadores de conteúdo externo. Ao direcionar mais público e visualizações para os vídeos de terceiros, contribuímos para aumentar a credibilidade e a importância do trabalho desses criadores, que ajudam milhares de pessoas todos os dias com seus conteúdos valiosos.

Os vídeos presentes neste site são de fontes externas, com os devidos créditos dados aos seus criadores. Os links para esses vídeos redirecionam diretamente para as plataformas originais, como o YouTube, garantindo que todos os direitos autorais sejam respeitados. Dessa forma, não há nenhum impacto negativo para os criadores dos vídeos, uma vez que o link utilizado no site funciona como um espelho, direcionando o usuário para a página do YouTube, onde as visualizações e a monetização acontecem normalmente, como ocorreria de qualquer outra maneira.

Ao apoiar e ampliar o alcance desses vídeos, o projeto oferece uma plataforma onde tanto os usuários quanto os criadores se beneficiam, criando um ciclo de aprendizado e reconhecimento mútuo.

Caso você seja o criador de algum conteúdo presente em nosso site e não deseje que ele seja exibido, por favor, entre em contato conosco pelo e-mail disponível no rodapé, e tomaremos as providências necessárias.

O público-alvo deste projeto é composto por pessoas que têm pouco ou nenhum conhecimento em tecnologia, mas que precisam de apoio para acessar o site e consumir o conteúdo, bem como por aqueles que desejam aprimorar suas habilidades tecnológicas. O site foi desenvolvido com foco na simplicidade e na acessibilidade, para que todos, independentemente de sua familiaridade com a internet, possam navegar e aprender com facilidade.

Em resumo, este projeto visa proporcionar uma experiência de aprendizado enriquecedora, ao mesmo tempo em que respeita os direitos autorais dos vídeos utilizados, e oferece uma plataforma intuitiva para todos os níveis de conhecimento tecnológico.
